# Glossary

- **Accrual P&L**: Performance view of profit based on sales and expenses, regardless of cash settlement timing.
- **Cashflow**: Liquidity view tracking actual cash in/out (settlements, expenses, wallet transfers).
- **Bridge Item**: Adjustment that explains the gap between accrual P&L and cashflow.
- **Director's Loan**: CEO â†’ Company transfer when commission is partially kept personally.
- **Import Batch**: Record in `import_batches` used to track file hash, row counts, and status.
- **Manual Mapping Wizard**: Fallback flow for mapping columns when auto-detection fails.
- **Performance Ads (Product/Live)**: Ads imports that include sales metrics and affect P&L.
- **Tiger Awareness Ads**: Ads imports without sales metrics; wallet spend only (cashflow-only).
- **RLS**: Row-Level Security, Supabase policy layer enforcing per-user data access.
- **Wallet Ledger**: Ledger of wallet transactions (TOP_UP, SPEND, REFUND, ADJUSTMENT).
